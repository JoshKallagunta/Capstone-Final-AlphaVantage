Welcome to the Stock Tracker

This application is used to track interday stock prices in intervals of 5,15,30, and 60 minutes. This will display in a table for easy viewing.

It uses the AlphaVantage API to get interday data. Future enhancements will include Simple Moving Average (SMA) and Relative Strength Index (RSI) tracking as well. Possibly the ability to graph the data.
